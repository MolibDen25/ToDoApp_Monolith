ToDoApp Monolith — README

C# console • Monolith • notes/*.txt

Коротко

Консольний застосунок для ведення нотаток у форматі ToDo з архітектурою Monolith. Дані зберігаються у папці notes у форматі .txt. Програма підтримує закреслення, редагування та видалення нотаток і рядків.

Структура проєкту

ToDoApp/
├── info/       # Папка з README
├── Program.cs  # Сама програма
└── notes/      # Тека збережених нотаток (створюється автоматично)

Запуск

1. Встановити .NET SDK (6.0+). Перевірити можна у терміналі:

dotnet --version

2. На Windows за допомогою ярлика ToDoApp або run_todo.bat.

Або у терміналі:

cd ToDoApp
dotnet run

Керування

Основні клавіші та команди:

  • 1 — вибрати пункт/нотаток
  • + — додати новий нотаток/рядок
  • ~№ — закреслити/розкреслити нотатку або рядок
  • -№ — видалити нотатку або рядок
  • Enter — зберегти редагування рядка
  • Esc — скасувати редагування
